# Das Auge Saurons

One does not simply walk into Mordor. Aber den Schnitzel Cat anklicken schon.

Live: **https://ancplua.github.io/sauron/**

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- [React Bits](https://reactbits.dev/backgrounds/evil-eye) `EvilEye` (WebGL via `ogl`)

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Auto-deploy via GitHub Actions auf `main` push.
